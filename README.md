#### **#  NeuroPad – Smart Notepad Application**

**#  Overview**

**NeuroPad** is a modern note-taking application developed as a group project for our university. It is designed to provide users with a simple, fast, and efficient way to create, organize, and manage notes, with additional support for cloud synchronization and rich text editing.

**# Features**

* Note Management
* Create, edit, and delete notes
* View all notes in an organized layout
* Sort notes by date created or modified

**# Rich Text Editing**

* Bold, italic, underline, and strikethrough
* Text alignment options
* Bullet and numbered lists
* Font size and color customization

**#  Cloud Sync**

* Sync notes with cloud storage
* View synced and unsynced notes separately
* Manual sync control for better flexibility

**#  User Interface**

* Clean and modern UI
* Responsive layout
* Smooth navigation experience


**# Technologies Used**

* Flutter – Frontend framework
* Dart – Programming language
* Flutter Quill – Rich text editor
* Backend used - Firebase

**# Installation \& Setup**

1\. Clone the repository:

```bash
git clone https://github.com/Rxyz72/NeuroPad.git
```

2\. Navigate to the project folder:
```bash
cd neuropad
```

3\. Install dependencies:
```bash
flutter pub get
```

4\. Run the app:
```bash
flutter run
```
